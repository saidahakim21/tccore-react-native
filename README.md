# tccore-react-native

CommandersAct's TCCore mobile library bridge to react-native

## Installation

```sh
npm install tccore-react-native
```

## Usage

```js
import { multiply } from 'tccore-react-native';

// ...

const result = await multiply(3, 7);
```
